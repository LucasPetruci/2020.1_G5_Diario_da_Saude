Introdução

  O BPMN (Business Process Model and Notation) é um padrão que utiliza uma representação gráfica para especificação de processos de negócio. Esta ferramenta foi desenvolvida para fornecer uma notação padrão facilmente compreendida para todos os envolvidos. Neste projeto, o BPMN será utilizado para detalhar a metodologia que será utilizada durante o desenvolvimento.

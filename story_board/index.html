Definição

  Storyboard é uma ferramenta que tem como objetivo apresentar uma ideia visualmente. Consiste de uma série de ilustrações ou imagens dispostas em sequência de forma a representar uma história.
  Na área de desenvolvimento de software, os storyboards são aplicados na identificação das especificações de uma parte do software. Normalmente apresentam as etapas mais importantes para uma boa experiência de usuário, de forma a serem de fácil entendimento e manutenção.
